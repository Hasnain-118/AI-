🤖 Hasnain AI Chatbot

A custom fine-tuned conversational AI chatbot built with HuggingFace Transformers and Streamlit.

 Features
- Fine-tuned DistilGPT2 on custom dataset
- JSON-based knowledge base (35+ instruction-response pairs)
- Streamlit UI with chat interface
- Real-time response generation

Run
streamlit run bot.py
