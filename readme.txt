Search Algorithm Visualizer
An interactive Jupyter Notebook that visualizes various pathfinding algorithms on a grid. Built with Streamlit for an engaging, hands-on learning experience.
🎯 Overview
This project provides a complete implementation of five classic search algorithms with real-time visual feedback. Users can interactively create mazes, run algorithms, and compare their performance side-by-side.

✨ Features
5 Search Algorithms: BFS, DFS, UCS, Greedy, and A*

Interactive Grid: Click to place walls, start, and goal positions

Real-time Visualization:

Blue cells = explored nodes

Yellow cells = final path

Green = Start, Red = Goal

Performance Metrics:

Path length

Nodes explored

Execution time

Comparison Mode: Run all algorithms simultaneously and compare results

Random Generation: Auto-generate walls with one click

🚀 Getting Started
Prerequisites
Python 3.7+

Jupyter Notebook
