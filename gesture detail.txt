🖐️ AI Gesture Control System
An advanced computer vision system that enables hands-free control of your computer using hand gestures. Built with MediaPipe, OpenCV, and Tkinter, this system allows you to control media playback, volume, scrolling, and capture screenshots through intuitive hand gestures.

🎯 Overview
This project implements a real-time hand gesture recognition system that translates hand movements into computer commands. Using Google's MediaPipe for hand landmark detection, the system recognizes specific finger configurations and triggers corresponding actions. Perfect for presentations, media control, accessibility applications, or just showing off!

Key Highlights
Real-time Gesture Recognition: 30+ FPS performance

Multi-platform Support: Windows, macOS, Linux

Intuitive Dashboard: Real-time feedback and gesture reference

Zero Training Required: Uses MediaPipe's pre-trained models

Lightweight & Efficient: Minimal resource usage

Core Gestures
Gesture	Action	Platform Support

☝️ Index (1 finger)	Volume Up	All
✌️ Two fingers	Volume Down	All
3 Fingers	Scroll Up	All
4 Fingers	Scroll Down	All
🖐 Open Palm (5 fingers)	Screenshot	All
👍 Thumb Only	Mute Toggle	All
